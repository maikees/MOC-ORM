MOC ORM is a fast object-relational mapper (ORM) for PHP.

That works with:
* SQL Server
* Postgress
* Mysql

## Install

`composer require moc-solucoes/moc-orm`

### Examples

You can find practical usage examples in [examples](https://github.com/maikees/moc-orm/tree/master/examples) path.
